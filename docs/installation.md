# Installation

## Stable release

To install s_spatioloji, run this command in your terminal:

```sh
uv add s_spatioloji
```

Or if you prefer to use `pip`:

```sh
pip install s_spatioloji
```

## From source

The source files for s_spatioloji can be downloaded from the [Github repo](https://github.com/gynecoloji/s_spatioloji).

You can either clone the public repository:

```sh
git clone https://github.com/gynecoloji/s_spatioloji
```

Or download the [tarball](https://github.com/gynecoloji/s_spatioloji/tarball/main):

```sh
curl -OJL https://github.com/gynecoloji/s_spatioloji/tarball/main
```

Once you have a copy of the source, you can install it with:

```sh
cd s_spatioloji
uv sync
```
